# Backend

Mobile Section React Native


## Run Locally

Clone the project

```bash
  git clone https://github.com/enganasalsaadi/almajid_mobile.git
```

Go to the project directory

```bash
  cd almajid_mobile
```

Install dependencies

```bash
  npm install
```

start the mobile ios 

```bash
  cd ios && pod install 
```

start the mobile ios 

```bash
  react native run-ios
```
